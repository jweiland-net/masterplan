============
Users Manual
============

Target group: **Editors**

#. Select a page where you want to insert masterplan

#. Insert a new content element of type: `Insert Plugin`

#. On tab `Plugin` select `Masterplan` from selectbox.

#. Confirm reload to load special input fields for our extension (FlexForm).

#. Save

Project Record
--------------

#. Select a Storage Page

#. Create a new record of type `Project`

#. Give it a title and maybe add some images and further information.

#. Enter a description which will be shown on configured detail page

#. By default you can insert up to 5 images to project record.

#. At tab `Categories` you can relate your project to a category.
